# Verificador de Pases — DGT Sección Discapacidad

## Cómo publicar en GitHub Pages

1. Subí todos estos archivos a tu repositorio de GitHub
2. En GitHub → Settings → Pages → Source: "Deploy from branch" → main → / (root)
3. En unos minutos tenés la URL pública

## Credenciales de Admin
- ID: ADMIN1
- Contraseña: ADMIN#9999

## Cómo actualizar el CSV cada 15 días
1. Entrá a Supabase → SQL Editor
2. Ejecutá: DELETE FROM pases;
3. Luego cargá los nuevos datos con los archivos SQL que genera Claude

## Instalación en celulares
1. El inspector abre la URL en Chrome (Android) o Safari (iPhone)
2. Chrome: menú (3 puntos) → "Agregar a pantalla de inicio"
3. Safari: botón compartir → "Agregar a pantalla de inicio"
